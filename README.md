## Symfony Vagrant.

Simple and sweet ansible provisioned vagrant environment for symfony. 

## Features (default)
* Ubuntu 18.04
* NGINX web server
* MYSQL database
* PHP 7.4
* Xdebug
* Redis

## Configuring
* Ansible role variables can be changed in the `provisioning/vars/vars.yml` file.
* Vagrant things in the `Vagrantfile`.
